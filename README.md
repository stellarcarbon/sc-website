# sc-website

Repository containing code for the <https://www.stellarcarbon.io> website.

Staging can be found on <https://test.stellarcarbon.io/>.

## Developer instructions

This is a NextJS project.

Install using

```sh
pnpm install
```

Run locally

```sh
pnpm run dev
```

Run unit tests

```sh
pnpm test
```

Run e2e tests

```sh
pnpm run e2e
```

Run e2e headless

```sh
pnpm run e2e:headless
```
